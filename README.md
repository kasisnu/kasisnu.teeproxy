kasisnu.teeproxy
=========

Use this role to get a teeproxy container running.

Requirements
------------

You will need to make sure Docker is running on the hosts this is run against.

Example Playbook
----------------

    - hosts: servers
      roles:
         - { role: kasisnu.teeproxy }
